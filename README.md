# 4D-Visualization-2

En las carpetas /4D_Proyeccion y /4D_Secciones usar `make`

Para usar otros datos copiar el contenido de un txt de /Datos a data.txt

Para crear un modelo de una función, modificar la función `f` en function.py y correr `python3 function.py >| data.txt`

`f` recibe una tupla de dos elementos y retorna una tupla de 2 elementos.

Hay ejemplos del programa corriendo en /Videos
